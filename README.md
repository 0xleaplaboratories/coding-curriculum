## ⚙️ Getting Started

Follow these steps to set up your learning session:

1. **Download Scratch** using the link for your platform above.
2. **Choose your level**, then download the corresponding game folder.
3. Inside the game folder, you'll find two file types:
   - `.sb3` — the main project file
   - `.sprite3` — individual learning session files (located in the `resources` folder)
4. **Open the `.sb3` file** in Scratch.
5. **Upload all `.sprite3` files** from the `resources` folder.
   - Each `.sprite3` file is one learning session containing guides and challenges.
   - All `.sprite3` files are linked together — they must all be uploaded before running.
6. **Run the game** once all `.sprite3` files have been uploaded. You're ready to learn!
